## Featured Project — QuantCluster

Using **PCA and K-Means clustering** to inductively identify latent behavioral bias profiles among fixed-income investors. Investigates the 2024 DALBAR anomaly — why did retail bond investors underperform the Bloomberg U.S. Aggregate by 232bps while simultaneously increasing fund contributions?

**Methods:** PCA · K-Means · scikit-learn · CRSP · Morningstar · DALBAR  
**Supervisor:** Prof. Apostolos Xanthopoulos, Lewis University
